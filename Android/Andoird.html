步骤 11 接下来转换为apk，也就是安卓的安装包 

将AAB文件转换为APK文件，可以使用Google官方的bundletool工具。这个工具需要JDK 8或更高版本，并通过命令行运行。下面是一些基本的步骤：

                1.首先下载bundletool工具，可以从Google官方仓库中下载最新版的bundletool.jar文件。

                2.打开终端窗口，导航到bundletool.jar的目录下。

                3.执行以下命令将AAB文件转换为APK文件：

                java -jar bundletool.jar build-apks --bundle=/path/to/your/app.aab --output=/path/to/output.apks
                其中，“/path/to/your/app.aab”是AAB文件的路径，“/path/to/output.apks”是输出APK文件的路径。在执行此命令后，bundletool将创建一个“output.apks”文件，其中包含应用程序的所有APK文件。

                4.将生成的APK文件解压缩到一个新的目录中。可以使用任何zip工具来完成这个步骤。例如，在Windows中，可以右键单击“output.apks”文件，选择“Extract All”选项。解压缩后，你会发现生成了多个APK文件。

                5.安装所有APK文件。你可以通过USB连接将设备与计算机连接，并使用adb命令安装这些APK文件。例如，执行以下命令来安装生成的“base-master.apk”文件：

                java -jar D:\666\bundletool.jar install-apks --apks=C:\Users\16201\Desktop\music1.apks
                adb
